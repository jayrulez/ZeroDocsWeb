Adding a Resource
=================

.. rst-class:: SearchDescription

The basics of how to **add a resource**.

To add a new resource to your project either:

* :ref:`Command<Commands2015>` : ``Add`` 
* In the Menu bar :menuselection:`Resource --> Add`
* Press the |TBAdd| Button

|pbr|

to open the Add Window:

|AddResource|

*The Add Resource Window*

|pbr|

* Select the resource type you wish to add
* Choose a name for your resource

|SelectResourceType|

* Click the Create button
