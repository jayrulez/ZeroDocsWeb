Adding & Removing Components
============================

.. _AddComponent2015:

Add a Component
---------------

Basics of how to add a component to an object.

* :ref:`Select the object<SelectObject2015>` to which you want to add the component

  |OWSelectTransform|

*Your object should be displayed when selected*

* In the Properties Window

  *   Click the |AddComponentButton| button 

This brings up the Add Component drop down:

  |AddComponentDropDown|

*   Start typing the name of the component you want to add: 

  |SpriteComponentSearch|

*Notice that component options appear in the drop down selection*

*   Click on desired component to add it to the object

|pbr|

In the Properties Window, we can see that component was successfully added:

|SpriteComponentAdded|

|pbr|

.. _RemoveComponent2015:

Remove a Component
------------------

Removing a component.

* :ref:`Select the object<SelectObject2015>` from which you want to remove the
  component
  
  |OWSelectTransform|
  
*Your object should be displayed when selected*

You may remove a component in either of two ways.

1. Using the **"X" Button**

* In the Properties Window
  
  * Click the "X"-shaped button on the right side of the Sprite component
  
  |RemoveComponentX|

2. Using the **Right-Click Submenu**

* In the Properties Window
  
  * Right-click the Sprite component
  
  * In the Right-Click Submenu
    
    * Click Remove
    
    |RemoveComponentRightClick|
