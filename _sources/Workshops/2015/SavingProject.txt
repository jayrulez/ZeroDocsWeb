Saving the Project
==================

.. rst-class:: SearchDescription

Saving the project will and should be done often. You can save your project in one of four ways:

* Press ``Ctrl+S``
* Press the Save button: |SaveIcon|
* :ref:`Command<Commands2015>` : ``SaveProject``   
* Select Menu :menuselection:`Project --> Save Project`
