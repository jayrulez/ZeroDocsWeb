Running the Game
----------------

.. rst-class:: SearchDescription

Running the game will and should be done often. You can run the game in one of four ways:

* Press ``F5``
* Press the Play button on the tool bar

|GameButtons|

*The standard green right-pointing triangle plays the game*

* :ref:`Command<Commands2014>` : ``PlayGame``   
* Select Menu :menuselection:`Project --> Save Project`
