MultiConvexMeshCollider

The MultiConvexMeshCollider is a collider meant for approximating concave objects with a series of convex pieces. The MultiConvexMesh itself is created in the MultiConvexMesh Editor, a powerful tool that allows the user multiple options for computing the mesh. While the MultiConvexMeshCollider is not limited to what objects it may be applied to, the editor is limited to only working with Sprites.

Common Uses

  • Approximating collision for concave objects using a series of convex pieces.

Creating a MultiConvexMeshCollider

The MultiConvexMeshCollider component may be added to any object as long as it has a Transform component, but it will not behave as intended until the Mesh property has been set to the MultiConvexMesh made specifically for the object in question. To create a new MultiConvexMesh, one must use the MultiConvexMesh Editor, a tool that allows for both automatic and manual creation of a MultiConvexMesh based on a given SpriteSource. The editor shows the exact boundaries being created; therefore, once the new MultiConvexMesh Resource has been completed, one only need to select it for the Mesh property and it will provide collision along those boundaries.
