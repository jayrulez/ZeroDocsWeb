HeightMapCollider

The HeightMapCollider is a component placed on a HeightMapObject. When using the HeightMapTool to create a HeightMapObject, the HeightMapCollider is automatically applied to the height map. Similar to the MeshCollider, the HeightMapCollider has no volume (i.e. if anything is in it, or spawns it, collision will not be resolved).
